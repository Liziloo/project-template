# project-template

A template repository for creating new personal apps based on your 
preferences and requirements.

## Getting Started

1. Clone this repository:
   ```
   git clone https://github.com/liz/liz-app-template.git my-new-app
   ```

2. Navigate to the project directory:
   ```
   cd my-new-app
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and visit `http://localhost:3000` to see your new app 
in action.
